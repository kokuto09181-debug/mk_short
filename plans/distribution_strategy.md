# 拡散戦略プラン

## 前提：コンテンツの性質

Wikipedia ベースの「知らなかった！」系ショートコンテンツ。  
**歴史マニアではなく、歴史に興味はあるがそこまで深くない層**に向けて発信する。

- ✅ 合う媒体：偶然おすすめで流れてきて「へえ」と思う人
- ❌ 合わない媒体：5ch歴史板・専門メディア・マニアコミュニティ（浅いと叩かれる）

ターゲットは**50代前後で歴史が好きなライト層**。

---

## コンテンツ構造

```
長編動画（10分・別パイプライン）
    └── 冒頭・見どころを切り抜き
         ↓
Shorts / リール（45〜60秒）  ← 現在自動生成中
         ↓
各プラットフォームへ同時投稿
         ↓
反応を観測 → 顧客がいる媒体に集中投資
```

## 媒体別・コンテンツ変換

| 元素材 | 変換先 | 加工量 |
|--------|--------|--------|
| 動画ファイル | LINE VOOM / ニコニコ / Instagram / Facebook | ほぼそのまま |
| 脚本テキスト | note / アメブロ / Yahoo!クリエイターズ | ほぼそのまま |
| note 記事 | SmartNews / Gunosy | 配信登録後は自動 |
| サムネイル + 一文 | X / LINE オープンチャット | テキスト追記のみ |
| 脚本テキスト | Voicy / stand.fm | 読み上げるだけ |

## 市場調査としての活用

複数媒体に投稿し **2〜4週間後に比較**する。

| 指標 | 確認項目 |
|------|---------|
| インプレッション | どの媒体で最も表示されたか |
| エンゲージメント率 | いいね・コメント・保存率 |
| YouTube 流入元 | どの媒体からYouTubeに来ているか |
| コメント内容 | どんな層が反応しているか |

→ 反応が出た媒体にリソースを集中する

### Phase 2 — note / アメブロへの自動投稿（工数: 小）

- [ ] Yahoo!クリエイターズ申請（手動・一度きり）

### Phase 4 — Instagram / Facebook リール（工数: 中）
- [ ] Meta for Developers でアプリ申請
- [ ] `src/meta_poster.py` 作成

### Phase 5 — 音声展開（工数: 小）

## 実装イメージ

```python
# pipeline.py
video_id = uploader.upload(...)        # YouTube（既存）

video_distributor.post(video_path)     # LINE VOOM / ニコニコ（Phase 3）
meta_poster.post(video_path, script)   # Instagram / Facebook（Phase 4）
```
