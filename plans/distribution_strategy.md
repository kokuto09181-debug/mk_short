# 拡散戦略プラン

## 概要

「知られざる偉人伝」チャンネルの視聴者獲得・拡散戦略。  
ターゲットは**50代以上の歴史好き**。YouTube Shorts・長編動画を起点に、考えうる全媒体へ展開し、どこに顧客がいるかを把握する。

---

## コンテンツ構造

```
長編動画（10分・別パイプライン）
    └── 冒頭・見どころを切り抜き
         ↓
Shorts / リール（45〜60秒）  ← 現在自動生成中
         ↓
各プラットフォームへ同時投稿
         ↓
反応を観測 → 顧客がいる媒体に集中投資
```

## 市場調査としての活用

複数媒体に投稿し、**2〜4週間後に以下の指標を比較**する。

| 指標 | 確認項目 |
|------|---------|
| インプレッション | どの媒体で最も表示されたか |
| エンゲージメント率 | いいね・コメント・保存率 |
| 流入元 | YouTube のトラフィックソース分析 |
| フォロワー増加速度 | どの媒体でファン化が起きているか |
| コメント内容 | どの層が反応しているか（年代・興味） |

→ 反応が出た媒体にリソースを集中する
